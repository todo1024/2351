课程介绍：

面向区块链及信息安全相关领域的一线开发人员和研究人员，本课程详细介绍了区块链技术发展历程，精讲对等网络、共识算法、密码学算法等区块链核心技术体系，重点分析区块链面临的传统安全问题和技术本身的安全问题；结合区块链应用场景，实战区块链开源平台。课程共分五个部分，具体为：区块链介绍、区块链核心原理与新技术、区块链的安全问题、区块链的评估技术、区块链平台部署、开发。
